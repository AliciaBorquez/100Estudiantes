# 100 Estudiantes Dijeron 🎓⚙️

Aplicación de competencia académica inspirada en el programa televisivo "100 Mexicanos Dijeron", desarrollada específicamente para la comunidad de **ITESCA**.

## 🚀 Descripción
El sistema permite gestionar dinámicas de preguntas y respuestas basadas en encuestas reales a estudiantes. Utiliza una arquitectura de **doble pantalla**:
1. **Panel de Control:** Para el administrador/maestro.
2. **Tablero Visual:** Para mostrar a los participantes y al público.

## 🛠️ Requisitos Previos
Antes de ejecutar el proyecto, asegúrate de tener instalado:
* [SDK de .NET 8.0](https://dotnet.microsoft.com/download/dotnet/8.0)
